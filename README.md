# SampleCustomView2
Building Android custom view from scratch by extending the View class.

## Requirements
* Android Studio 3.5 Canary 7
* Kotlin 1.3.30-eap-45
* Android Gradle Plugin 3.5.0-alpha07
* Gradle wrapper 5.2.1
* AAPT 2
